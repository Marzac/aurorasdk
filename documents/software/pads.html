Aurora software: gamepads library

This website regroups all documents you need to start developing games for Aurora.

General routines:

void padsPower(int enable)
int enable
enable power supply
Certain gamepad models need power supply. This function enables it on all ports.
int padsRead(int pad, int group);
int pad
Game pad index
int group
Game pad controls group
Read sensors state of one group. Groups are only available in multiplexed gamepads.
