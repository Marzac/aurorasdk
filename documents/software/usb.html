Aurora software: USB library

This website regroups all documents you need to start developing games for Aurora.

General routines:

void usbInit() Initialise serial over USB peripheral.

Configuration routines:

void usbSetRate(int baudrate)
int baud
Serial baudrate
Set the communication baudrate.

void * buffer
pointer to a receive buffer
int size
size of buffer in bytes
return

const void * buffer
pointer to a transmit buffer
int size
size of buffer in bytes
return

Write data to the USB transmit FIFO.
int usbWriteString(const char * string)
const char * string
pointer to a null-terminated string
return

Write a string to the USB transmit FIFO.
