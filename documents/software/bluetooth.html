Aurora software: Bluetooth library

This website regroups all documents you need to start developing games for Aurora.

General routines:

void blueInit() Initialise serial over Bluetooth peripheral.
void blueConfigure() Set default Bluetooth configuration (not mandatory). It configures:
  • Device name to: Aurora-SERIAL_NUMBER
  • Device password to: 1234
  • Baudrate to: 230400 bauds
Warning: a couple of seconds are needed to change bluetooth configuration.

Configuration routines:

void blueSetMode(int mode)
int mode
Bluetooth operating mode
Communication (BLUE_MODE_COM) or configuration (BLUE_MODE_AT)
Set the operating mode.
void blueSetRole(int role)
int role
Bluetooth communication role
Slave (BLUE_ROLE_SLAVE) or master(BLUE_ROLE_MASTER)
Set the communication role. Bluetooth must be in configuration (AT) mode.
void blueSetName(const char * name)
const char * name
Device name (null-terminated string)
Set the device name. Bluetooth must be in configuration (AT) mode.
void blueSetPassword(const char * pass)
const char * pass
Device password (null-terminated string)
A string made of four consecutive figures
Set the device password. Bluetooth must be in configuration (AT) mode.
void blueSetRate(int baudrate)
int baud
Serial baudrate
Set the communication baudrate.

Communication routines:

int blueRead(void * buffer, int size)
void * buffer
pointer to a receive buffer
int size
size of buffer in bytes
return
quantity of data read in bytes
Read data from the Bluetooth receive FIFO.
int blueWrite(const void * buffer, int size)
const void * buffer
pointer to a transmit buffer
int size
size of buffer in bytes
return
quantity of data written in bytes

int blueWriteString(const char * string)
const char * string
pointer to a null-terminated string
return
