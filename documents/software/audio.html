Aurora software: audio library

This website regroups all documents you need to start developing games for Aurora.

Raw audio:

General routines:
void rawAudioUpdate() Update raw audio subsystem. It performs channels modulations and must be called every frame.
Channels control:
void rawAudioInit(int channel)
int channel
channel index
Initialise channel properties and associated DSP peripherals.
void rawAudioStart(int channel)
int channel
channel index
Start channel playback.
void rawAudioStop(int channel)
int channel
channel index
Stop channel playback.
void rawAudioPitch(int channel, int note)
int channel
channel index
int note
MIDI note number
with 8 bits fractionnal part
Set the channel base pitch. Notes correspond to MIDI note numbers x 256. Fractionnal parts allows sub-semitone pitches.

int channel
channel index
int speed
modulation speed
int amount
modulation amount
Configure a pitch modulation effect.

int channel
channel index
int pw
Pulse-width ratio
range
Set the channel base pulse-width.

int channel

Configure a pulse-width modulation effect.

PCM audio:

General routines:
void pcmAudioUpdate() Update PCM audio subsystem. It services channels streaming buffers and must be called every frame.
Channels control:
void pcmAudioInit(int channel)
int channel
channel index
Initialise channel properties and associated DSP peripherals.
void pcmAudioStart(int channel, const FlashPage page, const FlashPtr buffer, int length)
int channel
channel index
const FlashPage page
sample buffer page in flash memory
page can be obtained with the FLASHPAGE(p) macro
const FlashPtr buffer
sample buffer pointer in flash memory
pointer can be obtained with the FLASHPTR(p) macro
int length
buffer length in samples
Start channel playback of an audio sample in Flash memory.
void pcmAudioStop(int channel)
int channel
channel index
Stop channel playback.
