Aurora software: internal flash library

This website regroups all documents you need to start developing games for Aurora.

Standard routines:

void flashLoad(const FlashPage page, const FlashPtr pointer, RamPtr buffer, int size)
const FlashPage page
data source page in flash memory
const FlashPtr pointer
data source pointer in flash memory
RamPtr buffer
destination pointer in standard sram
int size
size of the transfer in bytes
Load data from internal flash memory to standard sram.

Block based routines:

void flashBlockRead(const FlashPage page, const FlashPtr pointer, RamPtr buffer)
const FlashPage page
block page in flash memory
const FlashPtr pointer
block pointer in flash memory
RamPtr buffer
destination pointer in standard sram
Load a complete block (4K = 4096 bytes) from internal flash memory to standard sram.
void flashBlockErase(const FlashPage page, const FlashPtr pointer)
const FlashPage page
block page in flash memory
const FlashPtr pointer
block pointer in flash memory
Erase a complete block (4K) of internal flash memory, filling space with 0xFF.
Block erasing is mandatory before writting new data to internal flash.
void flashBlockWrite(const FlashPage page, const FlashPtr pointer, const RamPtr buffer)
const FlashPage page
block page in flash memory
const FlashPtr pointer
block pointer in flash memory
const RamPtr buffer
source pointer in standard sram
Store a complete block (4K = 4096 bytes) from standard sram to internal flash memory.
